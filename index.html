HTML & CSS Cheatsheet

Heading tags

Name Tag Purpose
h1 <h1> Highest level of heading element
h2 <h2> 2nd Highest level of heading element
h3 <h3> 3rd Highest level of heading element
h4 <h4> 4th Highest level of heading element
h5 <h5> 5th Highest level of heading element
h6 <h6> Lowest level of heading element

Formatting Elements

Name Tag Purpose
b <b> To make text bold
strong <strong> To highlight important text
i <i> To make text italic style
em <em> To show emphasized text
mark <mark> To make text marked
del <del> To show that text is removed or deleted
ins <ins> To show that the text has been inserted
sub <sub> To define subscript text
sup <sup> To define superscript text

Colors

Name Attribute Purpose
color color:red To set the color property for an element
rgba rgba() alpha values for transparency of colors in css
hsla hsla() alpha values for transparency of colors in css
hex #000000 color representation in hexadecimal notation
hsl hsl(200, 50%, 50%) color representation with hue(0 - 360), saturation(0% - 100%), light(0% - 100%)
